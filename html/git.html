Основные команды Git

  • git init – инициализировать новый локальный репозиторий (создаёт скрытую папку .git).
  • git clone <url> – клонировать удалённый репозиторий на локальную машину.
  • git status – показать текущее состояние рабочего каталога и индекс (staging area).
  • git add <file> – добавить файл(ы) в индекс для последующего коммита.
  • git commit -m "сообщение" – зафиксировать изменения в локальном репозитории с комментарием.
  • git branch – показать список веток (создать ветку: git branch <name>).
  • git checkout <branch> – переключиться на указанную ветку (или создать и переключиться: git checkout -b <name>).
  • git merge <branch> – влить указанную ветку в текущую.
  • git pull origin <branch> – получить изменения с удалённого репозитория и автоматически слить их с текущей веткой.
  • git push origin <branch> – отправить ваши коммиты на удалённый репозиторий.
